## Overview

The Health Monitor project is designed to demonstrate how health-related data can be displayed clearly and intuitively in a web interface. It focuses on **visual clarity, responsiveness, and modern UI design**, rather than real sensor integration or medical accuracy.

All values are **static or simulated**, making this project ideal for:
- UI/UX demonstrations
- Front-end practice
- Dashboard layout experimentation
- Portfolio showcase

---

## Features

These values are shown in a **card-based layout** for easy scanning.

### Modern UI Design
- Clean and minimal layout
- Card-based components
- Smooth spacing and typography
- Designed for readability, not clutter

### Responsive Layout
- Works on desktop, tablet, and mobile screens
- Uses Tailwind utility classes for flexible scaling
- No layout breaking on smaller screens

### Lightweight & Fast
- No frameworks (React, Vue, etc.)
- No backend
- No build tools
- Loads instantly in the browser
